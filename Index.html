KML to CSV Conversion Guide

Welcome to the KML to CSV conversion guide. Follow these steps to convert your KML file to a CSV file with coordinates:

Using QGIS

  1. Download and install QGIS: Get it from the QGIS official site.
  2. Load your KML file:
    • Open QGIS.
    • Go to "Layer" > "Add Layer" > "Add Vector Layer."
    • Select your KML file and click "Open."
  3. Extract coordinates:
    • Right-click on the layer in the Layers panel and select "Export" > "Save Features As."
    • In the "CRS" field, choose EPSG:4326 - WGS 84.
    • Select "Comma Separated Value [CSV]" as the format.
    • Set the file name and location.
    • In the "Geometry" section, set it to "AS_XYZ" to get X, Y, and Z values directly.

Conclusion

You now have a CSV file with the coordinates from your KML file. If you encounter any issues, feel free to ask for help!

Deployment with Git

To deploy this web page using Git:

  1. Initialize Git in your project folder with git init.
  2. Add your files with git add . and commit them with git commit -m "Initial commit with KML to CSV guide".
  3. Link your repo to GitHub with git remote add origin <your_repo_url>.
  4. Push your changes with git push -u origin main.
  5. Optionally, set up GitHub Pages in your repo settings for automatic deployment.
